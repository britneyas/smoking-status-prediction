# Smoking Status Prediction Using Bio-Signals

## Overview
A machine learning project to predict smoking status based on bio-signal data from 38,000+ records. Built as a final project for Machine Learning course at BINUS University.

## Dataset
- Source: Kaggle - Smoking and Drinking Dataset with Body Signal
- Features: 23 bio-signal features (age, height, weight, blood pressure, cholesterol, etc.)
- Target: Binary classification (smoker / non-smoker)
